# Задание 0.1

### Текст задачи
Опишите функцию nextch — модуль ввода-вывода. (Примечание: из-за отсутствия
анализатора, формирующего таблицу ошибок, создайте таблицу ошибок вручную.)



### Алгоритм решения

1. Составить ручную таблицу ошибок
2. Задать класс поля которого будут показывать информацию об ошибках такие как: Код ошибки,
сообщение ошибки, номер позиции в которой произошла ошибка
4. Разработать класс для посимвольного чтения файла и ввода исходного текста программы
5. Описать метод для открытия файла программы и обработки исключение,
вроде не найденного файла или ошибки чтения
7. Следующий класс называется NextCh и считатывает следующий символ программы,
до тех пор пока не достигнут конец исходного файла и обрабатывает исключения
8. Закрыть открытый исходный файл

# Задание 0.2

### Текст задачи

Разработайте набор тестов для тестирования модуля ввода-вывода.

Выполните тестирование модуля ввода-вывода.

### Алгоритм решения

1. Возьмем 6 файлов с разными исходными данными для тестов разных ситуаций чтения
2. Создадим код для открытия и посимвольного чтения наших файлов посредством ранее написанного метода NextCH
3. Закрыть и очистить(не обязательно) файлы
4. Получить результаты тестов всех 6 файлов в консоли и увидеть работу модуля ввода-вывода
